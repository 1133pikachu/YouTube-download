# YouTube-download
YouTubeの動画を指定してダウンロードできるやつです。

## 使用方法
### リポジトリのクローン
```
git clone https://github.com/1133pikachu/YouTube-download.git
```

### モジュールのインストール
```
npm i
```

### ダウンロードする動画,フォルダ名の指定
`download.js`のL65にURLを、L69に保存するフォルダ名を入れてください。

### 実行
```
node download.js
```
